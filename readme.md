# Fullstack Task - Flask + MongoDB Atlas

## Setup
1. Create a folder `fullstack_task` and put the files exactly as provided.
2. (Recommended) Create virtualenv:
   - `python -m venv venv`
   - `source venv/bin/activate` (Linux/Mac) or `venv\Scripts\activate` (Windows)
3. Install dependencies:
   - `pip install -r requirements.txt`
4. Ensure your `.env` has the correct MONGODB_URI and DB/collection names.
5. Run:
   - `python app.py`
6. Open http://localhost:5000

## Create ZIP
From parent folder:
`zip -r fullstack_task.zip fullstack_task`
